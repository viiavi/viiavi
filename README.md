# Hi, I'm vainavi (viiavi) 
### Embedded Systems Engineer 

I specialize in bridging the gap between **high-assurance security** and **resource-constrained embedded systems**. My work focuses on Post-Quantum Cryptography (PQC) migration, hardware-rooted trust, and real-time observability.

---

###  Featured Projects

####  [QuantumShield](https://github.com/viiavi/quantumshield)
**RTOS Middleware for PQC Migration**
*   Designed a Zephyr-based middleware for runtime classical-to-PQC migration.
*   Implements secure transition paths for IoT devices without hardware replacement.
*   Focuses on memory-efficient implementations of NIST-standardized PQC algorithms.

####  [RTOSTwin](https://github.com/viiavi/RTOSTwin)
**Embedded Runtime Observability Stack**
*   A FreeRTOS telemetry pipeline for STM32 using UART DMA.
*   Streams live task, heap, and CPU metrics to **Prometheus** and **Grafana**.
*   Built for high-performance V&V (Verification and Validation) in industrial RTOS environments.

####  [Sugarcane IoT](https://github.com/viiavi/sugarcane_iot)
**Decentralized Irrigation Consensus**
*   Developed a peer-to-peer validation protocol for multi-node STM32 sensor networks.
*   Replaced centralized cloud logic with edge-validated consensus to ensure reliability in low-connectivity environments.
